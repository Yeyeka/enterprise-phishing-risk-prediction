Preprocessor design:
- Linear models: numeric median imputation + StandardScaler; categorical most-frequent imputation + OneHotEncoder(handle_unknown='ignore').
- Gaussian Naive Bayes: numeric median imputation + StandardScaler; categorical most-frequent imputation + dense OneHotEncoder(handle_unknown='ignore'); ColumnTransformer uses sparse_threshold=0.0.
- Tree models: numeric median imputation only; categorical most-frequent imputation + OneHotEncoder(handle_unknown='ignore').

Feature engineering scope:
- Target binary mapping: No -> 0, Yes -> 1.
- Numeric missing values are handled with median imputation inside preprocessing pipelines.
- Categorical missing values are handled with most-frequent imputation inside preprocessing pipelines.
- Categorical predictors are one-hot encoded with handle_unknown='ignore'.
- Linear and GaussianNB preprocessing standardizes numeric features; tree preprocessing does not standardize numeric features.
- Two raw feature sets are maintained: full features and a reduced feature set excluding cybersecurity_awareness_score.
- No additional interaction or composite features are created because the source data already contains multiple behavior and awareness scores, several score fields are highly correlated, ad-hoc combinations could add redundancy, and later feature importance plus ablation testing is a cleaner way to judge feature value.
- IQR-based potential outliers are not removed because their rates are low and may reflect plausible employee behavior differences.
- SMOTE is not used because the positive class proportion is 37.31%, which is not severe class imbalance.

Data leakage prevention:
- This script never fits an imputer, scaler, or encoder on the full dataset.
- Validation calls fit_transform only on X_train and transform only on X_test.
- Fitted validation preprocessors are not saved.
- Future model training should keep preprocessors inside model Pipelines fitted on training folds only.
- Train/test split is performed before any future preprocessing fit.
- The test set is not used for fill values, feature selection, parameter tuning, or scaling statistics.
- The original CSV is read-only in this stage and is not modified.
